# Homey Motionblinds BLE

## Introduction
Homey Motionblinds BLE is a project designed to integrate and control Motionblinds using Bluetooth Low Energy (BLE) technology. This app allows users to manage their blinds through the Homey platform, providing a seamless smart home experience.

## Installation
To install the app, clone the repository and navigate to the project directory. Then, run the following command to install the necessary dependencies:
```
npm install
```

## Usage
After installing the dependencies, you can build and test the app using the following commands:
- To build the app, run: `npm run build`
- To test the app, run: `npm run test`

## Scripts
The project includes several npm scripts for common tasks:
- `build`: Compiles the TypeScript code using the TypeScript compiler.
- `test`: Runs the test suite using Jest.
- `lint`: Lints the codebase using ESLint.
- `format`: Formats the code using Prettier.

## Dependencies
The app relies on the following key dependencies:
- `crypto-js`: For cryptographic operations.
- `luxon`: For date and time manipulation.
